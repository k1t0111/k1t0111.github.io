# 优化主页 + 文章排序

## 改动

### 1. `index.html` — 主页改色 + 丰富描述

**配色：** 去掉所有黑色，换成温暖自然色调
- 主色：深松绿 `#5b8a72`（avatar渐变、按钮、顶部渐变条）
- 点缀：暗红 `#c0392b`（dot、em标签）
- 背景：暖白 `#faf9f7`
- 文字：深灰 `#2c2c2c`（非纯黑）

**avatar：** 绿色渐变圆形，带柔和阴影

**顶部横条：** 三色渐变 `#5b8a72 → #c0392b → #d4a76a`

**个人描述扩充为 3 段：**
1. AI Agent 安全方向 — 智能体攻防、工具调用风险、Prompt 注入
2. 渗透/红队/CTF — Web安全、内网渗透、权限提升、横向移动
3. 个人理念 — 记录学习，帮助同行

**标签卡改为信息卡：** 三个带边框的卡片（方向/领域/技术栈）

**按钮：** 绿色背景白字，悬停加深 + 阴影

### 2. `articles.html` — 有图文章排前面

**排序逻辑改动：** render() 中
```
有 cover → 排前面（按日期倒序）
无 cover → 排后面（按日期倒序）
```

不改其他样式。

## 涉及文件
- `index.html` — 重写 CSS + HTML 结构
- `articles.html` — 仅改 JS render() 排序逻辑

## 验证
- 浏览器打开 index.html 确认无黑色元素、描述完整
- 打开 articles.html 确认有封面图的文章排在前面
