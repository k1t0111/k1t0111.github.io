匹配到例如.php 之后则会舍弃 php 后面的 url 路径
也就是说 index.php 和 index.php/1.php 效果是一样的

preg_match('/config.php/*/i′, /i', /i′, _SERVER['PHP_SELF'])
这个匹配的是 config.php///////....
但是 highlight_file (basename ($_SERVER ['PHP_SELF']
这个语句 也是利用了 SERVER 这种 获取 url
同时 basename 是截取最后一个路径的文件 但是如果识别 到不可识别的字符例如
% ff 那么我们就可以 再往前 截取一个文件
