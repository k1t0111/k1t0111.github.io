### [](#创建数据库 "创建数据库")创建数据库

#### [](#简单方式 "简单方式")简单方式

```sql
CREATE DATABASE 数据库名;
```

#### [](#稍稍难 "稍稍难")稍稍难

> _**目的是 查看 是否你的名字重复**_

```sql
CREATE DATABASE IF NOT EXISTS mydatabase;
```

#### [](#带参数 "带参数")带参数

```sql
CREATE DATABASE [IF NOT EXISTS] database_name  [CHARACTER SET charset_name]  [COLLATE collation_name];-- 看看实力CREATE DATABASE mydatabase    CHARACTER SET utf8mb4    COLLATE utf8mb4_general_ci;
```

## [](#删除数据库 "删除数据库")删除数据库

```sql
DROP DATABASE <database_name>;        -- 直接删除数据库，不检查是否存在或DROP DATABASE [IF EXISTS] <database_name>;
```

## [](#使用 "使用")使用

```sql
show databases;use  相对应数据库名字;
```

## [](#修改表-alter "修改表 alter")修改表 alter

### [](#添加列 "添加列")添加列

```sql
ALTER TABLE table_nameADD COLUMN new_column_name datatype;
```

### [](#修改列的数据类型 "修改列的数据类型")修改列的数据类型

```sql
ALTER TABLE TABLE_NAME  MODIFY COLUMN column_name new_datatype;
```

### [](#修改列名 "修改列名")修改列名

```sql
ALTER TABLE table_nameCHANGE COLUMN old_column_name new_column_name datatype;
```

### [](#删除列 "删除列")删除列

```sql
ALTER TABLE table_nameDROP COLUMN column_name;
```

### [](#添加-PRIMARY-KEY "添加 PRIMARY KEY")添加 PRIMARY KEY

```sql
ALTER TABLE table_nameADD PRIMARY KEY (column_name);
```

### [](#添加-FOREIGN-KEY "添加 FOREIGN KEY")添加 FOREIGN KEY

```sql
ALTER TABLE child_tableADD CONSTRAINT fk_nameFOREIGN KEY (column_name)REFERENCES parent_table (column_name);
```

### [](#修改表名 "修改表名")修改表名

```sql
ALTER TABLE employees  RENAME TO staff;
```
