#### [](#2-三级跳 "2. 三级跳")2\. 三级跳

```text
lcx_slave  该管道一侧通过反弹方式连接代理请求方，另一侧连接代理提供主机。lcx_tran   该管道，通过监听本地端口接收代理请求，并转交给代理提供主机。lcx_listen 该管道，通过监听本地端口接收数据，并将其转交给目标网络回连的代理提供主机。    通过组合lcx类别管道的特性，可以实现多层内网环境下的渗透测试。下面是一个三级跳的本地测试例子。。。./ew -s rcsocks -l 1080 -e 8888./ew -s lcx_slave -d 127.0.0.1 -e 8888 -f 127.0.0.1 -g 9999./ew -s lcx_listen -l 9999 -e 7777./ew -s rssocks -d 127.0.0.1 -e 7777数据流向为   IE -> 1080 -> 8888 -> 9999 -> 7777 -> rssocks 
```
