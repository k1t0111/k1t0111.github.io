# [](#查询 "查询")查询

### [](#单表查询 "单表查询")单表查询

```sql
SELECT column1, column2, ...FROM table_name;与SELECT * FROM table_name;**参数说明：**- **column1, column2, ...**：要选择的字段名称，可以为多个字段。如果不指定字段名称，则会选择所有字段。- **table_name**：要查询的表名称。-- 参数说明DISTINCTSELECT DISTINCT column1, column2, ...FROM table_name;
```
