#### [](#2-初始化本地仓库 "2. 初始化本地仓库")2\. 初始化本地仓库

```shell
hexo init rm -rf node_modules package-lock.jsonnpm install --force
```

此时已经安装好了hexo，并且初始化完毕。

#### [](#3-相关命令 "3. 相关命令")3\. 相关命令

```shell
hexo cl 删除hexo g 生成hexo s 本地搭建hexo d 自动部署
```

#### [](#4-更换主题 "4. 更换主题")4\. 更换主题

下载 主体到theme文件夹下，设置主config 更换主题

接下来如果想要真的自己喜欢的界面，那么就需要知道你选择的主题的架构。一般来讲hexo的主config文件只是一个大概的框架，但是主题的config才是可以diy的地方。
