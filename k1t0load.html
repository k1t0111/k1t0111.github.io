基础工具链
检查 curl HTTP 客户端
curl --version
检查 Python3 脚本运行时
python3 --version
检查 SSH 安全远程连接
ssh -V
检查 dig DNS 工具
dig -v
检查 openssl TLS / SSL 库
openssl version
Python 标准库依赖
检查 json 模块 JSON 编解码
python3 -c "import json; print('ok')"
检查 socket 模块 底层网络通信
python3 -c "import socket; print('ok')"
检查 ssl 模块 TLS 加密支持
python3 -c "import ssl; print('ok')"
网络连通性
DNS 解析 github.com 域名解析验证
dig +short github.com A
HTTPS 握手测试 TLS 连接验证
curl -sf -o /dev/null -w "%{http_code}" https://github.com
服务器连通性 39.96.170.193:5555
curl -sf http://39.96.170.193:5555/v1/models -H "Authorization: Bearer $PUBLIC_SECRET" -o /dev/null
API 端点延迟 GitHub API 响应时间
curl -sf -o /dev/null -w "HTTP %{http_code} · %{time_total}s\n" https://api.github.com
仓库可达性
验证仓库可访问 k1t0111/blog
curl -sf https://api.github.com/repos/k1t0111/blog | python3 -m json.tool
获取默认分支 仓库主分支名称
curl -sf https://api.github.com/repos/k1t0111/blog | python3 -c "import sys,json; print(json.load(sys.stdin)['default_branch'])"
获取最新提交 最近一次 commit 信息
curl -sf "https://api.github.com/repos/k1t0111/blog/commits?per_page=1" | python3 -c "import sys,json; c=json.load(sys.stdin)[0]; print(c['sha'][:7], c['commit']['message'].split(chr(10))[0])"
